# Modulotech Tap

## How do I install these formulae?

`brew install modulotech/tap/<formula>`

Or `brew tap modulotech/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "modulotech/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
